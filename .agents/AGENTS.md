# CP Playbook Project Charter

This document provides high-level project context and philosophy for any AI agent joining this codebase.

## 1. Project Purpose
The **CP Playbook** is a structured competitive programming training resource. It is designed to prepare students and developers for:
- **LCPC** (Libyan Collegiate Programming Contest)
- **ICPC** (International Collegiate Programming Contest)
- General competitive programming growth.

The project is initiated and maintained by the **SHU Developers Club** community.

---

## 2. Reference & Syllabus Source
The core curriculum structure and roadmap are adapted from the **YouKn0wWho Academy** platform.
- We have taken their comprehensive roadmap and restructured it to prioritize topics based on importance, creating a tailored, step-by-step learning progression.

---

## 3. Roadmap Philosophy
Competitive programming is a vast universe. This playbook uses an **Importance-driven** progression to make learning efficient:
- **High Importance**: Core topics that students must master first. These are the primary targets.
- **Medium Importance**: Topics to study next (can be safely skipped or deferred until High-importance topics are fully understood).
- **Low Importance**: Advanced or niche topics. These can be skipped initially, studied for fun, or tackled after mastering High and Medium-importance topics.

---

## 4. Programming Language Priority
- **Python**: The primary programming language used by the students.
- **Adapted Resources**: Syllabus resources and code snippets must prioritize Python.
- **C++**: Will be integrated as an additional language in the future, but is NOT the focus right now.

---

## 5. Content Rollout Strategy
- **Basics Category**: The initial focus. We are uploading custom handwritten explanations and practice sets slowly.
- **Placeholder Pages**: For topics where custom content is not yet prepared, we display styled "Under Construction" placeholder pages pointing to the YouKn0wWho Academy syllabus as a reference.
